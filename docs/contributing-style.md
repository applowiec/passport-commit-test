# Styl kontrybucji

## Commity
Stosujemy [Conventional Commits](https://www.conventionalcommits.org/):
- feat: nowa funkcja
- fix: poprawka błędu
- docs: dokumentacja
- chore: porządki
- inne: refactor, test, ci, build

## Kod
- Spójne wcięcia (2 spacje)
- Linia max. 100 znaków
- README i CHANGELOG zawsze aktualne
